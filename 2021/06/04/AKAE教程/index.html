AKAE教程

1.new composition

名字:Crossfire 1920*812 5s 字体Astrolab 网上下载macOS字体集安装即可 文本工具 输入5 放大居中

2.预合成 Layer pre-compose

Name : Text Edit 后面可以修改字 move all attributes into new composition

3.三维效果 Layer-layerstyle-inner glow

Settings-BlendMode-Normal

Color->Black

Technique->precise

size->6

Layer-pre-compose Beval Profile move all attributes to new composition

4.凹凸效果

layer new Solid 起名glass

Color B 50%

右上角-effects presets -CC glass 拖到solid上上

左上panel-Surface-Bump Map-select Beval Profile softness ->0 Displacement->0 很硬的斜角

5.透明背景

把Glass图层放在Beval下面

打开轨道遮罩 设为alpha matte

6.添加三维效果 Light AE Light

a.Layer-new-Light set to point cast shadows soft

b.拖到左上方

c.Light option Intensity 367%

d.cmd+d 复制一个放在右下 亮度 调280
