PS2019官方教程

一.基本技能

1.第一节PhotoShop教程

a.眼睛打开关闭图层

b.移动图层

c.图层不透明度

d.图像-调整-色相饱和度

e.污点修复画笔

f.置入嵌入对象,CMD+T

2.第二节处理多个图层

a.调整图层大小

b.调整图层顺序

3.选择图像的一部分

a.快速选择工具

b.CMD+D 取消选择

c.调整-亮度饱和度

4.使用图层蒙版向图像中添加对象

a.建立蒙版-选中蒙版-选中画笔-设置前景色为黑色-用黑色画笔涂画,可以隐藏不想要的部分,只留下铅笔

b.用白色画笔涂画,可以重新显示隐藏的图像

5.保存图像

a.魔棒,选中白色背景,DEL,可以变为透明,CMD+D取消选择,可以保存为PNG,保留透明部分

二.修复照片

1.调整配色

a.图像-调整-亮度对比度

b.图像-调整-自然饱和度,自然饱和度暗色的强度,饱和度所有颜色的强度

2.冷色和暖色图像

a.图像-调整-照片滤镜 冷却滤镜 浓度35

3.裁剪和拉伸图像

a.裁剪工具 裁剪完可以旋转再裁剪

4.调亮深色区域

a.图像-调整-高光阴影 45%

5.移去小对象

a.污点修复工具,可以去掉污点

三.制作创意效果

1.自定义黑白

a.图像-调整-黑白 调亮或调暗特定颜色 黄色

2.复制小对象和纹理

a.仿制图章工具 Option 选中对象 拖动绘制 会以目标为中心仿制 一次只能复制一次

3.选择图像的一部分

a.快速选择工具 Option取消选择

b.调整 亮度对比度

4.替换背景

a.快速选择工具 调整工具大小 硬度

b.选中铅笔

c.添加蒙版 直接把没选中的区域变为透明

d.打开另一种背景就替换了背景

5.向图像中添加人物

a.选中图层 快速选择工具 画笔10 100%

b.选中人物 添加蒙版 蒙版可以隐藏选中的区域以外所有的区域

c.打开背景 实现添加人物

6.创建光照效果

a.打开光照照片 混合模式设为颜色减淡 不透明度设为50

7.应用滤镜

a.滤镜-转换为智能滤镜

b.滤镜-风格化-油画 没有opencl 没有这个选项

c.选中预览复选框

d.描边样式 拖到1.7

e.描边清洁度 55

四.绘画

1.选择颜色进行绘制

a.吸管工具 自动添加到前景色

b.从色板选择一种颜色

c.点击前景色 选择颜色

d.点击切换前景色和背景色

e.点击黑白快速设置按钮

2.使用笔刷工具绘画

a.选择笔刷工具 大小60 硬度30 不透明度 流量

b.吸管工具 选择颜色

c.绘制鱼鳃

PS知识

1.调整文本框

cmd+T

2.文字外发光

图层样式

3.移动工具

没选中自动选择,要先选中图层,才能移动

选中自动选择,会自动选择图层

鼠标放在目标周围外边一点就会出现旋转指示

一般不选中显示变换控件,用cmd+T

4.Option点击眼睛只看某一图层

5.图层可以编组

6.历史记录 可以撤回

7.图层锁定,剪切会填充背景色

8.矩形选择

选中区域,右键,通过拷贝的图层,可以创建选区的新图层 CMD+J

Shift 加选区

Option,减选区

Option+Shift 交叉选区

羽化可以让粘贴更顺滑

变换选区,可以调节

Option+Del 填充前景颜色

Ctrl+Del 填充背景色

填充前,先创建新图层

如果是白色就是透明,填充没效果

椭圆选择,可以Shift选正圆

单行选框工具-可以用来画线 一个像素

9.套索选择

羽化可以先设置,也可以建选区后右键羽化

大概选区可以用

多边形套索-画直边

磁性套索工具-自动识别颜色,可以画曲线

磁性套索可以按Option,变成多边形套索

10.Option+滚轮 放大缩小

11.选择过程中,按住空格键,可以移动

12.选中并遮住

a.平滑

b.移动边缘

c.调整边缘画笔 可以自动计算边缘颜色 如毛发能去掉多余的颜色

AKAE教程

1.new composition

名字:Crossfire 1920*812 5s 字体Astrolab 网上下载macOS字体集安装即可 文本工具 输入5 放大居中

2.预合成 Layer pre-compose

Name : Text Edit 后面可以修改字 move all attributes into new composition

3.三维效果 Layer-layerstyle-inner glow

Settings-BlendMode-Normal

Color->Black

Technique->precise

size->6

Layer-pre-compose Beval Profile move all attributes to new composition

4.凹凸效果

layer new Solid 起名glass

Color B 50%

右上角-effects presets -CC glass 拖到solid上上

左上panel-Surface-Bump Map-select Beval Profile softness ->0 Displacement->0 很硬的斜角

5.透明背景

把Glass图层放在Beval下面

打开轨道遮罩 设为alpha matte

6.添加三维效果 Light AE Light

a.Layer-new-Light set to point cast shadows soft

b.拖到左上方

c.Light option Intensity 367%

d.cmd+d 复制一个放在右下 亮度 调280
